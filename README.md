# 法玛全天候趋势回归动量指数风控策略

## 项目简介

本项目是一个基于 Python 的量化交易系统，实现了**全天候趋势回归动量指数风控策略**。该策略结合了动量因子（Momentum）、趋势回归分析以及指数级别的风险控制，旨在在捕捉市场趋势的同时有效控制回撤风险。系统支持多种交易接口（QMT、同花顺/EasyTrader）和数据源（QMT、东方财富、JoinQuant 等）。

**作者**: 法码量化

## 核心功能

*   **动量策略 (Momentum)**:
    *   基于加权线性回归计算对数价格的斜率，得出年化收益率。
    *   结合 $R^2$（拟合优度）计算动量得分：$Score = Annualized\ Returns \times R^2$。
    *   每日计算股票池中 ETF/股票的动量排名，持有排名靠前的标的。
*   **双重风控机制**:
    1.  **个股/ETF 趋势风控**: 当标的价格低于指定的移动平均线（如 N 日均线）时，强制卖出或不买入。
    2.  **指数级风控**: 监控大盘指数（如上证指数 000001.SH）。当大盘指数跌破其移动平均线时，对特定标的（如创业板 ETF）执行清仓保护。
*   **多平台支持**:
    *   **QMT (极速交易系统)**: 支持国金 QMT 等券商接口。
    *   **同花顺 (THS)**: 通过自动化 UI 操作（EasyTrader 变种）进行下单。
*   **自动化运行**:
    *   支持定时任务（Schedule），自动在指定时间（如 09:35）进行调仓。
    *   自动发送交易通知（支持 QQ 邮箱、企业微信、钉钉）。

## 环境依赖

本项目运行需要 Python 环境（建议 Python 3.8+），并依赖以下主要库：

*   `pandas`, `numpy`, `scipy` - 数据处理与科学计算
*   `akshare`, `pytdx`, `pywencai` - 数据获取
*   `easyquotation` - 实时行情
*   `pywinauto`, `pyautogui` - 自动化控制（用于同花顺交易）
*   `schedule` - 定时任务
*   `matplotlib`, `mplfinance` - 绘图

安装依赖：
```bash
pip install pandas numpy scipy akshare easyquotation pywinauto pyautogui schedule matplotlib mplfinance pytdx
# 或者运行提供的批处理文件
# 安装第三方库原始.bat
```

此外，如果使用同花顺自动化交易，可能需要安装 [Tesseract-OCR](https://github.com/tesseract-ocr/tesseract) 用于验证码识别。

## 配置说明

### 1. 系统配置 (`分析配置.json`)

此文件配置交易环境、账户信息和路径。

```json
{
    "交易系统": "qmt",                  // 选择交易系统：'qmt' 或 'ths'
    "交易数据源": "qmt",                // 数据源选择
    "qmt路径": "D:/国金QMT交易端模拟/userdata_mini", // QMT userdata_mini 路径
    "qmt账户": "5500xxxx",             // QMT 资金账号
    "同花顺下单路径": "C:/同花顺软件/同花顺/xiadan.exe", // 同花顺下单程序路径
    "识别软件安装位置": "C:/Program Files/Tesseract-OCR/tesseract", // OCR 路径
    "交易时间段": 4,
    "交易开始时间": 9,
    "交易结束时间": 14
    // ... 其他账户设置
}
```

### 2. 策略配置 (`策略配置.json`)

此文件控制策略的具体参数。

```json
{
    "策略名称": "trendindex",
    "调仓时间": ["09:35"],             // 每日执行调仓的时间点
    "股票池": ["518880", "513100", "159915"], // 参与轮动的 ETF/股票代码
    "股票池名称": ["黄金ETF", "纳指100", "创业板100"],
    "动量天数": 25,                    // 计算动量的窗口窗口（天）
    "风险控制线": 10,                  // 个股趋势均线参数（如 10 日均线）
    "指数代码": "000001.SH",           // 风控参考指数
    "指数风控线": 10,                  // 指数风控均线参数
    "指数风控标的": ["159915"],        // 受指数风控影响的标的
    "买入数量": 1,                     // 每日持仓的最大标的数量
    "交易模式": "金额",                 // '金额', '数量', 或 '百分比'
    "下单值": 10000,                   // 单次下单金额或数量
    "是否开启策略隔离": "是"            // 是否只操作策略配置中的股票，不影响账户其他持仓
}
```

## 使用方法

1.  **配置环境**: 确保 Python 环境及依赖库已安装，Tesseract-OCR 已配置（如需）。
2.  **修改配置**: 根据实际情况修改 `分析配置.json` 和 `策略配置.json`。
3.  **启动 QMT/同花顺**: 
    *   如果是 QMT 模式，需先启动 QMT 客户端并登录。
    *   如果是同花顺模式，需启动同花顺并保持独立交易端打开。
4.  **运行策略**:
    直接运行项目根目录下的主程序：

    ```bash
    python 法玛全天候趋势回归动量指数风控策略.py
    ```

    程序启动后会进行账户检查，然后进入等待状态，到达 `调仓时间` 时自动执行交易逻辑。
